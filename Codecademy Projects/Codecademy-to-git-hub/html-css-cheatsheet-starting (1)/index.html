Fundamentals of HTML Cheatsheet
Body Headings and Sub Headings Text Blocks Text Emphasis Line Breaks Lists Images
Any visible content should be placed within opening and closing tags. Uses tags ranging from H1-H6. Provides titles Uses: P, span, and div to specify text or blocks Uses em and strong tags to emphasize text. Line breaks are created using br. Lists are created using ol and ul. Use img or video to link an image or video to a source.
